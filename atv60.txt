//Funcionamento web - protocolo cliente x servidor

O funcionamento web se dá por mensagem enviada pelo cliente através do software específico utilizado pelo mesmo, que envia a mensagem ao servidor, que com o seu software lhe permite receber a mensagem, interpreta-la e devolver a resposta para o cliente.
